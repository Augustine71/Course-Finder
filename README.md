# Project Title

Course Finder

## Description

A react application to navigate through available courses.

Link: https://course--finder.herokuapp.com/

## Getting Started

### Instructions

Clone the repo & then

### `npm install`

To install all the dependencies

### `npm run dev`
To run the application
